github注册

注册过程我就不详细说了，注册完成后会进入这样一个界面，点击加号，选择new repository，如图：


接着你只需填写Repository name，这里注意一下，必须是yourname.github.io，yourname就是前面的owner，如我的：



别的就可以不用管了，直接点create repository，现在就完成了。

工具准备

以下工具都可以在我的百度云盘找到，当然你随意从哪儿下载都可以，只是省得麻烦而已。网盘地址：click here，密码：uouy。

node.js安装

这里我就不详细说了，搜官网直接下载就好了，安装配置一路默认就好。我已经从百度搜完了，你可以从这直接下载，32位click here或64位click here，官网都是。

git安装

同上，一路默认即可，32位，64位。

上面两个都下载完成后，用管理员身份打开cmd，分别输入node -v，npm -v,git –version,如果和我的差不多，说明你上述的都安装成功了。如图：

hexo安装

//以后所有命令都在git bash中输入(cmd有的会执行不了)，虽然会很卡，下面就不在赘述了。

hexo安装必须安装完上面的才成，
在任意空白位置右击，选择git bash here，输入npm install -g hexo -cli，如图：



然后新建一个文件夹，名字位置随意，例如我在E盘新建一个MyBlog文件夹，进入文件夹，右击选择git bash here，分别输入hexo init,npm install,这个hexo init命令可能会等很长时间，如果时间很长还没有完成，关了重新再输就好了。最终效果如图：





现在hexo就安装完成了，当然你要还有兴趣的研究更多的话，可以参看hexo官网。

git github相关联：

在git bash中分别输入：git config –global user.name “github用户名”

效果如图：



git config –global user.email “邮箱”

密钥生成

输入：ssh-keygen -t rsa -C “邮箱”，接着连续点三次回车就好了，当然你也可以自己指定保存位置，密码，第一次是让输入保存位置，后两次是密码。一般默认即可。如图:



接下来复制rsa_pub中的文本并打开github,点击头像再点击settings，点ssh key and gpg key,将复制的内容添加进key中就好了如图：





最后点击add ssh key就完成了。

上传github

更改_config.yml文件，找到deploy，并进行下面更改，如图：



先将开启的服务关闭，按Ctrl+c,然后分别输入hexo g,hexo d。这时你执行hexo d会报一个错：如图：



这是因为现在最新的版本还需要下载个东西，输入npm install hexo-deployer-git –save,下载完成后重新输入hexo g,hexo d就好了。大功告成，现在已经上传成功了。
你可以访问：https://yourname.github.io

如我的：
